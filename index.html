Coffee Shop

Mi primer proyecto web, una landing page para una cafetería ficticia. El objetivo fue aplicar los conocimientos iniciales de desarrollo web, trabajando con HTML, CSS, Sass y Bootstrap. El diseño es responsive y presenta secciones típicas de una página comercial, como menú, ubicación y formulario de contacto. Aprendí a estructurar contenido semántico, usar un framework CSS y organizar estilos con preprocesadores.
